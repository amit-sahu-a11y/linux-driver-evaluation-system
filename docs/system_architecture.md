#### ðŸ” b. Static Analysis Module (`evaluation/static_analysis.py`)
- Parses C code for:
  - Security issues (buffer overflows, race conditions)

#### ðŸ“Š c. Metrics & Scoring (`evaluation/metrics.py`)

---

### 3. **Reporting Engine**
- Consolidates the evaluation into a single `JSON` report.
- Output folder: `reports/`
- Example: `evaluation_output.json`
